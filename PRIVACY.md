# Privacy Policy

Harness Loom is a local Codex plugin and repository toolset. It does not include built-in analytics, telemetry, or background data collection.

The plugin reads and writes files only in repositories where you invoke it. Network activity occurs only when you explicitly run commands that use remote git hosts, package registries, or other third-party services.
